# RoadMap — ZapDNS

## Версия 0.1 (MVP) — Android (цель: 4–6 недель)

### Цели
- Рабочий DPI-bypass на базе byedpi
- Интеграция своего DoH
- Базовые профили
- Автозапуск + удобный UI

### Задачи

- [ ] Форк/изучение ByeDPIAndroid + ByeByeDPI
- [ ] Сборка byedpi под Android (NDK, несколько ABI: arm64, armv7, x86_64)
- [ ] Базовый VpnService + запуск byedpi
- [ ] Простой DoH клиент + перехват DNS
- [ ] Jetpack Compose UI (Home + Settings + Profiles)
- [ ] 4 предустановленных профиля
- [ ] DataStore + Room для хранения
- [ ] BootReceiver + Foreground Service
- [ ] Quick Settings Tile
- [ ] Тестирование на реальных блокировках (РФ)
- [ ] Сборка debug APK + инструкция по установке

**Релиз**: Debug-версия для тестировщиков (через GitHub Releases)

## Долгосрочные идеи (v2+)

- **Kotlin Multiplatform** — общие модули DPI + DNS для Android + Desktop + возможно iOS
- Умный режим: приложение само определяет, когда нужен агрессивный bypass
- Интеграция с **NekoBox** / **Matsuri** как плагин (byedpi-neko уже существует)
- Поддержка **WireGuard** как альтернативного fallback
- Сообщество стратегий (репозиторий стратегий DPI)
- Анонимная краудсорс-статистика заблокированных доменов (с согласия пользователя)
